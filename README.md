# GANFromAtoZ for Day2Night Project
This repository tries to generate synthetic night images using various GAN methods.

## GAN models
- CycleGAN
- DCGAN
- DiscoGAN
- Pix2pix
- Vanilla GAN  
- VAE
- WGAN
